## Cross-Reviewer Themes

Several issues were spotted independently by multiple reviewers:

1. **Error state inconsistency** — Maren (4 patterns), Dex (F12 missing retry), Tomás (no standard component)
2. **Dark mode missing** — Maren (dead dark: prefixes), Priya (dispatch rooms are dim)
3. **Tab implementation** — Maren (3 different patterns), Priya (ARIA missing on all)
4. **Status color confusion** — Dex (5 statuses = green), Priya (color-only differentiation)
5. **Form UX gaps** — Dex (no inline validation), Tomás (no dirty-close on employee forms)
6. **Table features unused** — Dex (no sorting/pagination enabled despite DataTable supporting both)
